# 🎤 Text to Audio App

A simple web app that converts text into speech using the browser’s built-in Web Speech API.

## 🚀 Live Demo

https://texttoaudio--vaibhavambrisha.replit.app

## ✨ Features

* Convert text to speech
* Adjust pitch & volume
* Play, pause, and stop audio
* Download audio

## 🛠️ Tech

HTML, CSS, JavaScript

## ▶️ Run Locally

Open `index.html` in your browser.
